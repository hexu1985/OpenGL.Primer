Chapter 9 programs:

bezier.c: interactive program for drawing cubic bezier curves. Points are entered
with mouse. Curves are begun and terminated with keyboard.

curves.c: generalization of bezier.c that allows same data to specify cubic bezier,
spline, or interpolation curves.

bteapot.c: displays Utah teapot using bezier surfaces. Control date are in
vertices.h and patches.h
