Chapter 7 programs:

bit.c: demo of writing a bitmap and use of XOR

ppmdisplay.c: displays a ppm image stored in a file (robot2.ppm is there for testing)
demoing writing pixel maps.

readpixels.c: demos reading pixels from the display. The image displayed is a constant color
but slightly different values will be read back if the display used dithering.

image4.c: rough program for displaying a luminance image.
